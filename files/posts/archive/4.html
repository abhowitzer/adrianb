Het beeld is breed joh

[13/04/17 ~20:00]

Shieet. Wat lopen nadenken over één of andere implementatie voor een iets logischere manier van editing van deze teksten. Kwam eerst uit op Dillinger, een markdown editor+exporter. Dat loopt op Node.JS. En het is me allemaal heel erg vreemd. Dat lijken me een paar stappen te ver, te vroeg.

Maar mijn idee blijft: ik wil een makkelijke manier om deze teksten te schrijven op een intuïtieve wijze (betreft markup), dat ook nog eens volautomatisch kan. Nu is Javascript me vreemd, maar Bash niet. En wat heb ik? No shit een server met Linux en bash. Dus gebruiken wat ik kan. Ennnnnn, blijkbaar kan je shell scripts draaien op een webserver?

Web is vreemd. Zó breed, zó diep. Heb er al wel wat glimpsen van opgevangen over de jaren, maar het gaat véél verder dan ik vermoedde. NPM is verdomme een repository waarmee je javascript-shizzle automagisch installeert om het dan in een website te implementeren? Wut, bestond (deftig) package management niet enkel op Debian/consoorten?

Ik voel me een beetje ontdekkingsreiziger zo. Dacht dat ik er allemaal wel wat van wist, maar elke dag is een nieuw continent.
